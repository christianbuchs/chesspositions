# chesspositions
Upper Bound for the number of possible chess positions

The dissertation finds a newer lower bound for the number of chess positions and analysis further improvements. It builds on the results of Chinchalkar and Tromp and proves their calculations mathematically (as the proves provided by both of them were very short and not rigorous). This dissertation was my bachelor's dissertation and got the final grade 1.0, which is equivalent to 100%. The dissertation was submitted in September 2023, without the use of AI (except for grammar synonyms and spelling and latex formatting) (but not used in any quantitative regard). The English translation was done by ChatGPT in February of 2025. In doubt, the German version should be considered. As part of the dissertation I translated Tromp's code from Haskell to C (already including a lot of runtime improvements) to make it comparable to my code, as my code wouldnt be able to run in a reasonable time in Haskell.

My code is provided in "mycode.c". The C-Translation of Tromps code is provided in "kopievontromp.c"
